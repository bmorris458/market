Welcome to the Market!

To test the system:

First, feel free to click any of the 'Query' links below, and see that they do not return any values. Then, add one of the users and click the user query links again. See that they return only the expected user. Click to add the other user and then see 'Query all users' return both. Note the empty Set for each user. Click to add a subscription for each user, and then 'Query all users' again to see that the subscriptions have been added.

Now, add the items, but do not add any tags yet. Check the subscriptions for user 'Ben'. There should be no notifications, as currently notifications are only checked against tags, and no tags are generated automatically when an item is added. Now, add the tag to 'Black Widow' (the comic Ben has as a subscription). Click 'Check notifications for Ben' and see the notification show up. Check notifications again and see that none are returned, as the notification has already been consumed. Then, remove 'Black Widow', and check the notifications again to see the notification that the item has been sold.

Feel free to play with any combination of the links (or check out the URIs to see the pattern for adding and removing things and modify them, then query the system to see the changes--hopefully it's reasonably intuitive).

Also, to demonstrate recoverability, use the stop link to shut down the application, then return to the console and `sbt run` the application again. It will replay any actions taken and restore the state of the system prior to shutdown.
